PurposeÂ¶

  • The purpose of this notebook is to merge APOE and pTau217 data on available records to explore genetic and biomarker associations across diagnostic and demographic groups.
  • As well identifying population-specific risk factors and metrics related to AD among Hispanics.

Aim 1: APOE Îµ4 Prevalence and ImpactÂ¶

Objectives

  1. Assess Îµ4 prevalence and impact as a risk factor among Hispanics vs. non-Hispanic Whites (NHWs).
  2. Compare Îµ4 carriership between Hispanics and NHWs across three diagnostic groups:
    • Normal Cognition
    • Mild Cognitive Impairment (MCI)
    • Dementia
  3. Assess Îµ4 as a risk factor for cognitive diagnosis (Normal Cognition = control) while accounting for cardiovascular comorbidities.
  4. Evaluate interaction effects â€” determine if APOE Îµ4 associations differ significantly by ethnicity.
    • Adjust analyses for age, sex, and education.
    • Reference tables and figures will illustrate stratified comparisons.

Aim 2: pTau217 Distributions Across Diagnostic and Ethnic GroupsÂ¶

Objectives

  1. Visualize pTau217 distributions across Normal, MCI, and Dementia groups â€” separately for Hispanics and NHWs (modeled on NCRAD-style distribution plots).
  2. Test for significant differences in pTau217 levels across diagnostic categories.
  3. Compare inter-group variability â€” does pTau217 vary more in one ethnicity than another?
  4. Explore whether BMI and cardiovascular diagnoses act as pTau217-modifying factors.

Aim 3: APOE Îµ4 and pTau217 AssociationÂ¶

Objectives

  1. Examine how Îµ4 presence influences pTau217 levels within and across diagnostic groups.
  2. Quantify whether APOE genotype effects on pTau217 are consistent across cognitive strata.
  3. Reference figures will depict Îµ4-by-diagnosis interactions in pTau217 concentration.

This notebook will contain code to harmonize and integrate APOE, pTau217, and NACC-derived data sources.
It will also include preliminary analyses that support data quality review, dataset finalization,
and readiness for full statistical modeling and publication-ready analysis.

# normalization and clean up function.
def clean_df(df: pd.DataFrame) -> pd.DataFrame:
    df = df.copy()
    obj_cols = df.select_dtypes(include="object").columns

    # strip only real strings
    df.loc[:, obj_cols] = df.loc[:, obj_cols].apply(
        lambda s: s.map(lambda x: x.strip() if isinstance(x, str) else x)
    )

    # replace exact "." with NA only in object columns
    # This comes the NACC PORTAL where missing values are "."
    df.loc[:, obj_cols] = df.loc[:, obj_cols].replace(".", pd.NA)

    df = df.infer_objects(copy=False)
    return df

# create a visit date column in each dataframe
def date_formatter(df: pd.DataFrame, form_name: str) -> pd.DataFrame:
    df = df.copy()
    df["VISIT_DATE"] = pd.to_datetime(
        df["VISITYR"].astype(str)
        + "-"
        + df["VISITMO"].astype(str).str.zfill(2)
        + "-"
        + df["VISITDAY"].astype(str).str.zfill(2)
    )
    df.drop(columns=["VISITMO", "VISITDAY", "VISITYR"], inplace=True)
    # rename to unique name based on form
    df.rename(columns={"VISIT_DATE": f"{form_name.upper()}_Date"}, inplace=True)
    return df


# process all dataframes to add VISIT_DATE column with unique names
form_mapping = {"a1": a1, "b1": b1, "d1": d1, "d2": d2}

for form_name, df in form_mapping.items():
    form_mapping[form_name] = date_formatter(df, form_name)

# Unpack back to individual variables
a1, b1, d1, d2 = form_mapping["a1"], form_mapping["b1"], form_mapping["d1"], form_mapping["d2"]

NACCMBIÂ¶

  • Allowable codes 10.0 â€“â€‰â€‰100.0
  • 888.8 = Unknown or not assessed
  • 88.8 = Unknown or not assessed
  • -4 = Not available: UDS form submitted did not collect data in this way, or a skip pattern precludes response to this question

Description/derivation Body mass index is derived using variables height (pounds) and weight (inches). The standardized calculation used is as follows: BMI = ( weight (lbs) Ã— 703) Ã· height (in) 2 If height or weight is missing or unknown, then naccbmi =888. If Form B1 was not submitted, then naccbmi = -4.

    if row["demented"] == 1:
        return "Dementia"
    elif row["mci"] == 1:
        return "Mild Cognitive Impairment"
    elif row["normcog"] == 1:
        return "Normal Cognition"
    else:
        return None
